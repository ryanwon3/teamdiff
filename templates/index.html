Same-lane opponents, ranked by average early gold difference — plus gold curves per matchup

0 lists everyone you’ve faced in the same lane with usable timeline data. Raise this to hide noisy one-off matchups.

Δ is opponent gold minus yours at the latest sample ≤15 min, averaged per opponent champion.

For each game, “lane” is Riot’s assignment (TOP / MIDDLE / BOTTOM / UTILITY / JUNGLE) — so a mid Irelia is compared only to the enemy mid. The table lists every such opponent champion, sorted by average gold gap at ≤15 min (opponent minus you); default is smallest gap first. Use the sort control to flip direction. Needs ingested timelines and MATCHUP_QUEUE_ID in .env if you filter by queue.
